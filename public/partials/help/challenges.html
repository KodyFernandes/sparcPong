Challenges

Prerequisites

Before you can issue, resolve, or forfeit challenges, you need an account. So pick a user name and sign up. You must be logged in before any of the below options are available.

Issuing Challenges

  • Issue a challenge by clicking the board button, "Challenge."
  • An example is shown below.


Reporting Results

  • You must be logged in to report results.
  • View the profile of either player and click the "Report" button.
  • An example is given below.
  • Depending on your challenges, different options will appear.


Resolving Challenges

  • Click the "Resolve" button that appears.
  • Report the number of games each player won.

Forfeiting Challenges

  • Click the "Forfeit" button that appears.
  • Only the challengee can forfeit.
