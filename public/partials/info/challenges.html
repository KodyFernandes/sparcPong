Prerequisites

Before you can issue, resolve, or forfeit challenges, you need an account. So pick a user name and sign up.

Issuing Challenges

  • Issue a challenge by clicking the board button, "Challenge."
  • Cannot challenge yourself. Duh!
  • Cannot have multiple unresolved challenge between two players.
  • Can only issue challenges during week days.
  • Can only issue challenges to players above your rank.
  • Can only issue challenges to players within 1 tier.
  • Cannot issue challenges before unresolving incoming challenges.
  • Cannot have more than 1 outgoing challenge.

Resolving Challenges

  • Report the match by clicking the profile button, "Resolve."
  • Report the number of games each player won.
  • The match cannot end in a tie.
  • The match cannot be resolved if it has expired.

Forfeiting Challenges

  • Forfeit a challenge by clicking the profile button, "Forfeit."
  • Only the challengee can forfeit.
  • Challenges may be forfeited before the challenge expires.
  • Challenges must be forfeited after the challenge expires.
